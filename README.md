# Elix Bot

Discord-бот с модульной системой: логирование, ранги, музыка, автомодерация.

## Требования

- Docker и Docker Compose
- Discord Bot Token ([Discord Developer Portal](https://discord.com/developers/applications))

Для запуска без Docker дополнительно нужны Python 3.11+ и FFmpeg.

---

## Быстрый старт (Docker)

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd bot.ds
```

### 2. Создать файл окружения

```bash
cp .env.example .env
```

Открыть `.env` и вставить токен бота:

```env
DISCORD_TOKEN=ваш_токен_бота
PREFIX=!
```

### 3. Запустить

```bash
make up
```

или без Make:

```bash
docker compose up -d
```

---

## Управление через Make

| Команда | Действие |
|---|---|
| `make up` | Запустить контейнер в фоне |
| `make down` | Остановить и удалить контейнер |
| `make restart` | Перезапустить бота без пересборки |
| `make logs` | Следить за логами в реальном времени |
| `make build` | Пересобрать образ без кэша |
| `make update` | Обновить бота (см. ниже) |
| `make shell` | Открыть bash внутри контейнера |

## Обновление бота

Когда в репозиторий пришли новые изменения (новые модули, фиксы), выполните:

```bash
make update
```

Команда делает три шага:

```
git pull                          # 1. Загружает изменения из репозитория
docker compose build --no-cache   # 2. Пересобирает образ с нуля
docker compose up -d              # 3. Запускает обновлённый контейнер
```

После этого в терминале появятся последние 50 строк логов — вы сразу увидите, поднялся ли бот без ошибок.

Данные базы данных (`data/`) и логи (`logs/`) хранятся в Docker-томах и **не теряются** при обновлении.

Если нужно только перезапустить без обновления кода:

```bash
make restart
```

## Запуск без Docker

### 1. Установить зависимости

```bash
pip install -r requirement.txt
```

На Linux/macOS также установите FFmpeg:

```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
```

### 2. Запустить

```bash
python run.py
```

## Структура проекта

```
bot.ds/
├── bot/
│   ├── __init__.py

├── dashboard.py         # Ког автомодерации и команд модераторов
├── db.py                # Утилиты для работы с базой данных
├── logging_cog.py       # Ког логирования событий сервера
├── music_cog.py         # Музыкальный ког (YouTube, очередь треков)
├── rank_cog.py          # Система рангов и XP
├── run.py               # Скрипт запуска
├── Makefile             # Удобные команды для управления контейнером
├── Dockerfile
├── docker-compose.yml
└── requirement.txt
```

## Команды бота

### Музыка

| Команда | Описание |
|---|---|
| `!play <запрос>` | Добавить трек в очередь |
| `!stop` | Остановить воспроизведение |
| `!skip` | Пропустить трек |
| `!pause` / `!resume` | Пауза / Возобновить |
| `!queue` | Показать очередь |
| `!np` | Текущий трек |
| `!volume <0-100>` | Громкость |

### Ранги

| Команда | Описание |
|---|---|
| `!rank [@user]` | Статистика и уровень пользователя |
| `!leaderboard [N]` | Таблица лидеров |
| `!voicetime [@user]` | Время в голосовых каналах |

### Модерация (требует прав)

| Команда | Описание |
|---|---|
| `!clear [N]` | Удалить N сообщений (макс. 100) |
| `!kick @user [причина]` | Выгнать участника |
| `!ban @user [причина]` | Забанить участника |
| `!forbidden add/remove/list` | Управление фильтром слов |
| `!settings` | Настройки панели управления |
