# Elix Bot

Discord-бот с модульной системой: логирование, ранги, музыка, автомодерация.

## Требования

- Python 3.10+
- FFmpeg (для музыкального модуля)
- Discord Bot Token

## Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd bot.ds
```

### 2. Создать файл окружения

Скопируйте `.env.example` в `.env` и заполните переменные:

```bash
cp .env.example .env
```

Содержимое `.env`:

```env
DISCORD_TOKEN=ваш_токен_бота
PREFIX=!
```

Токен бота можно получить на [Discord Developer Portal](https://discord.com/developers/applications).

### 3. Установить зависимости

```bash
pip install -r requirement.txt
```

На Linux/macOS также установите FFmpeg:

```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg
```

### 4. Запустить бота

```bash
python run.py
```

Бот создаст директории `data/` (база данных) и `logs/` автоматически при первом запуске.

---

## Запуск через Docker Compose

### 1. Создать `.env` файл (см. выше)

### 2. Запустить

```bash
docker compose up -d
```

### 3. Посмотреть логи

```bash
docker compose logs -f
```

### 4. Остановить

```bash
docker compose down
```

Данные базы данных сохраняются в томе `bot_data` и не теряются при перезапуске.

## Структура проекта

```
bot.ds/
├── bot/
│   └── main.py          # Точка входа, загрузка когов
├── data/                # SQLite база данных (создаётся автоматически)
├── logs/                # Логи бота (создаются автоматически)
├── dashboard.py         # Ког автомодерации и команд модераторов
├── db.py                # Утилиты для работы с базой данных
├── logging_cog.py       # Ког логирования событий сервера
├── music_cog.py         # Музыкальный ког (YouTube, очередь треков)
├── rank_cog.py          # Система рангов и XP
├── run.py               # Скрипт запуска
├── requirement.txt      # Python зависимости
├── Dockerfile
└── docker-compose.yml
```

## Команды бота

### Музыка
| Команда | Описание |
|---|---|
| `!play <запрос>` | Добавить трек в очередь |
| `!stop` | Остановить воспроизведение |
| `!skip` | Пропустить трек |
| `!pause` / `!resume` | Пауза / Возобновить |
| `!queue` | Показать очередь |
| `!np` | Текущий трек |
| `!volume <0-100>` | Громкость |

### Ранги
| Команда | Описание |
|---|---|
| `!rank [@user]` | Статистика и уровень пользователя |
| `!leaderboard [N]` | Таблица лидеров |
| `!voicetime [@user]` | Время в голосовых каналах |

### Модерация (требует прав)
| Команда | Описание |
|---|---|
| `!clear [N]` | Удалить N сообщений (макс. 100) |
| `!kick @user [причина]` | Выгнать участника |
| `!ban @user [причина]` | Забанить участника |
| `!forbidden add/remove/list` | Управление фильтром слов |
| `!settings` | Настройки панели управления |
